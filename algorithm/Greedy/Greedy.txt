Greedy(탐욕법)

- “현재 상황에서 가장 최선이라고 생각되는 선택을 반복적으로 하는 방식”
- 영어로는 Greedy → 욕심쟁이처럼 매 순간 가장 좋아 보이는 선택을 한다는 뜻
- 최종적으로 **전체 문제의 최적해(최대한 좋은 결과)**를 구할 수 있는 경우에 적용 가능


장점
- 구현이 간단하고 직관적
- 탐색이나 DP보다 속도 빠름

단점
- 모든 문제에 적용 가능하지 않음
- 최적해가 아닌 근사해만 나올 수 있음


그리디 선택 속성(Greedy Choice Property)
- 매 단계에서의 최선 선택이 전체 최적해로 이어짐

최적 부분 구조(Optimal Substructure)
- 큰 문제의 최적해가 작은 문제 최적해로 구성됨


접근방법
1. 문제를 작은 단계로 나눈다
2. 각 단계에서 현재 순간 최선의 선택을 한다
3. 반복해서 전체 문제를 해결한다
4. 단, 항상 최적해가 나오는 것은 아님 → 그리디 선택 속성과 최적 부분 구조가 있어야 함
